# ✧ Tabuada ✧

Projeto educacional interativo desenvolvido para auxiliar no aprendizado da tabuada de 1 a 10 de forma visual, simples e intuitiva. A interface utiliza cores e elementos gráficos para tornar o estudo mais leve e acessível, especialmente para estudantes iniciantes.

## ✦ Objetivo

Facilitar a compreensão da tabuada

Tornar o aprendizado mais visual

Apoiar o ensino de matemática básica

Unir educação e desenvolvimento front-end

## ✧ Conteúdos

Tabuadas do 1 ao 10

Organização visual por blocos

Resultados claros e de fácil leitura
